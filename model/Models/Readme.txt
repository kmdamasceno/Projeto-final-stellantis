* Rename "BD_PAR_SS.mdl" to "BD_PAR.mdl".
* Extract "advisor-2003-00-r0116.zip".
* Copy "BD_PAR.mdl" to ./advisor/models and substitute the existing file.
* Open matlab and load to workspace "./Workspaces/bd_par_udds_base_hot.mat".
* Add advisor folder and subfolders to path.
* Open "BD_PAR.mdl" in simulink.
* Then, you can run the simulation directly in simulink.
* If you want, run the simulation in advisor.
* To do this, run "advisor" command in MATLAB.
* Setup the simulation in advisor toolbox, selecting "parallel" vehicle and "cyc_udds" drive cycle. The click in "run".
